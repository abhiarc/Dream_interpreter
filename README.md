# Dream Interpreter (Streamlit)

AI-based dream interpretation app built with Streamlit and the OpenAI API.

## Features
- Interpret a dream using multiple schools (Ancient + Modern approaches).
- Library tab explaining each interpretation tradition.
- Child-friendly interpretation style (per system prompt rules).

## Tech stack
- Python
- Streamlit
- OpenAI API
